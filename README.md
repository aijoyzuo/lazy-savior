# Lazy Savior

A recommendation quiz web app built with **React** and **TypeScript**.

This project helps users discover suitable recommendations across different categories such as books, movies, and food through interactive quiz flows.

---

## Tech Stack

- React
- TypeScript
- JavaScript
- HTML5
- CSS3
- Node.js
- npm

---

## Features

- Book recommendation quiz
- Movie recommendation quiz
- Food recommendation quiz
- Interactive quiz flow
- Recommendation result pages
- Modular component-based architecture
